Version 2.1.0

Codec Info will now detect and inform users about known issues with some codecs. If you know about an issue that's not being displayed, please send a feedback email about it!

Added a notification for the user if getting a codec list is impossible due to an Android platform bug.

Added more known color formats from Broadcom, MediaTek and Samsung.

Getting a supported profile level for VP9 codecs should be now much more accurate on Android 6.0 and older.

Bugfixes and general improvements.

Version 2.0.0

Added a new tab with DRM types (and their properties) supported by the device.

Added a search functionality - search for info in a current list of codecs/DRM or codec/DRM properties!

UI overhaul providing improvements for larger screens, landscape orientation and dark mode.

Bugfixes and general improvements.

Version 1.11.0

Improved hardware acceleration detection for some older Qualcomm codecs.

(Android 10 and newer) Aliases for other codecs are now hidden by default; this can be changed in settings.

Added color formats from newer Qualcomm SoCs.

Bugfixes and general improvements.

Version 1.10.0

Added support for Android TV!
NOTE: Sharing codec info and sending feedback from Android TV are currently unavailable.

Added color formats from Panasonic/SNI.

Added features and codec profiles from Android 11.

About app dialog was greatly reworked for devices with Android 5.0 and newer. It now also includes a changelog and licenses for used libraries.

App can now be installed and moved to an SD card.

Codecs will now note if they are software only and their provider.

Codec features supported on Android 5.0 and newer will now indicate if they are required to use.

Setting light/dark app theme should be now more reliable on some LG and Xiaomi devices.

Immersive mode is now a toggable option from settings.

Improved detection of some hardware-accelerated codecs.

Improved detection of input channel count and sample rates for ALAC.

Improved detection of WMA codecs.

Bugfixes and general improvements.
